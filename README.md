# Morse_code_project
The project displays Morse code, and then the corresponding letter for said Morse code is displayed, as well as a corresponding buzzer and LED. 
Equipment:
-8051(Micro_Controller)
-LCD screen
-LED
-Resistors:(~220 or above)
-Active Buzzer
-Two Buttons

Schematic: On personal website coming soon!!!

Description:
This project utilizes the 8051 as well as timers and interrupts, and processes Morse code and determines the difference between a dot and a dash. Doing this by seeing how long the button is pressed. Then, with the Enter key, the user can press Enter and see the letter they entered in Morse code. Every time the Morse code button is pressed their is a buzzer that is set off with an LED that turns on and is able to remain on for the duration that the user is pressing the button. 

Summary:
With a Morse code button, the user is able to press it and hold it to make dots and dashes, and then use an enter button to display the letter that the user created. With a corresponding LED and Buzzer that is determined by the Morse code button and how long or how little it is pressed.
